About this project

MYCMS is a multi-tenant content management system. The backend is implemented in Java (Java 21) and built with Gradle. The frontend is a Next.js application written in TypeScript. The reports linked on this page are generated by the project's CI pipeline and published to the repository's Pages output (coverage and test reports are produced by the frontend and backend build steps respectively).

This page is intended as a quick index to those generated reports and not as the primary project documentation. For development, building, and deployment instructions, see the repository README and the backend/frontend folders.
